HyperText Markup Language

HTML (HyperText Markup Language) is the standard markup language for creating Web pages.

It's not a programming language, it's just a plain text in a browser.

To write an HTML you need:

  1. Element content:
    • Tells the browser to do something.
  2. Attribute contents:
    • Tells the browser how to do something.

HTML needs 2 types of elements:

  1. Block-Level elements:

    Example:

    <div id="header">

    <p>Article heading</p>

    <p>Home : Articles : Article one</p>

    </div>

  2. Inline elements:

    Example:

    <div id="footer">

    <p>Author:

    <span class="author-name">Nik</span></p>

    </div>

HTML lists include:

  1. Unordered list:

    Example:

    <ul>

    <li>Coffee</li>

    <li>Milk</li>

    </ul>

  2. Ordered list:

    Example:

    <ol>

    <li>Coffee</li>

    <li>Milk</li>

    </ol>
