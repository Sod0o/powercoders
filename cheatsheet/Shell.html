Shell Cheatsheet

Machine (physical Computer) runs an Operating System, which manages access to everything. To interact with the OS, we use shell. So the OP knows which programs to run.

The shell is just another program.

There are 2 types of shell:

  1. Command line
  2. Graphicalical

Shell command line

It's also called:

  • terminal
  • command window
  • command prompt
  • command line interface (CLI)

You can access it through (windows):

  • Press Windows key, then type cmd

Anatomy of the command line

It consists of prompt, command, option and argument.

                           Option
                            |
        C:\Users\Sodaba> dir \w *.png
                    |        |        |
                 Prompt  Command  Argument
        _______________________________________________
                         Option
                           |
        macbook:~ suman$ ls -l *.png
                |        |        |
              Prompt  Command  Argument
